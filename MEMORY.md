# MEMORY.md - 安全准则

## 准则（铁律）

1. **操作权限**：严禁删除、修改、剪切、覆盖任何本地文件/文件夹，仅开放只读权限，仅允许在专属目录新建文件
2. **路径限制**：仅可访问专属目录，禁止访问系统目录、桌面、文档、下载及公司客户/订单/合同/木工机械业务数据等所有敏感目录
3. **执行约束**：任何文件操作必须先向虾指挥确认，未收到明确允许不得执行；收到删除/修改指令直接拒绝
4. **数据安全**：禁止读取、复制、传输客户信息、外贸数据、公司机密、账号密码、API密钥等敏感内容，禁止上传本地数据至外网
5. **异常处理**：访问禁止路径或违规操作时，立即终止任务并同步告知虾指挥
