### 5.4 ROI计算方法

#### ROI计算公式

```
ROI = (预期收入 - 投入成本) / 投入成本 × 100%
```

#### 投入成本明细

| 成本项 | 金额（元） | 占比 |
|--------|-----------|------|
| Alibaba年费 | 30,000 | 15% |
| Made-in-China年费 | 15,000 | 8% |
| 社媒运营人力 | 30,000 | 15% |
| 官网建设 | 20,000 | 10% |
| 视频制作 | 25,000 | 13% |
| 展会参加 | 80,000 | 41% |
| **总计** | **195,000** | **100%** |

#### 预期收益计算

```
成交客户数 × 平均订单金额 = 预期收入
15个客户 × 50,000元 = 750,000元
```

#### ROI预测

```
ROI = (750,000 - 195,000) / 195,000 × 100% = 285%
```

**注意:** 以上为乐观估计，实际ROI受产品定价、竞争态势、汇率等因素影响
